# Техническое задание — Pulse

## Название
**Pulse** — простой музыкальный хостинг (MVP).

## Функции
1. Регистрация и авторизация.
2. Загрузка аудио и обложки.
3. Каталог, поиск, HTML5-плеер.
4. Избранное.
5. **Плейлисты** — создание, добавление своих треков, публичные/личные.
6. REST API: треки, пользователи, плейлисты.
7. Валидация данных в модуле `validation/`.
8. Экспорт каталога в CSV.

## Технологии
Python, Flask, SQLAlchemy, SQLite, Bootstrap, Flask-WTF, Flask-Login.

## Не входит в MVP
iTunes и прочие сторонние API, навык Алисы, сложный дизайн.
